IIIF Image Rotator

A simple little tool to provide rotated IIIF URLs. Input a valid IIIF image URL, then click and drag on the icon to rotate. Image shown may appear larger or smaller than the IIIF URL indicates. This is to ensure small images can be rotated easily.

Disclaimer: Rotation is not supported by all IIIF servers. Please see the IIIF API documentation for more information.
